--------------------------------------------------------------------------------
HOW TO GET INSTAGRAM IMAGES WORKING
--------------------------------------------------------------------------------

Option 1: Add Instagram Access Token (Recommended)
--------------------------------------------------
1. Go to https://developers.facebook.com/
2. Create a new app
3. Add "Instagram Basic Display" product
4. Add your Instagram account as a test user
5. Generate an access token
6. Add to your .env file:
   IG_ACCESS_TOKEN=your_token_here
7. Run: npm run fetch-instagram

Option 2: Upload Images Manually
--------------------------------
1. Add your images to: /server/data/images_for_upload/
2. Supported formats: .jpg, .jpeg, .png, .gif, .webp
3. Run: npm run fetch-instagram
4. Images will be uploaded to Cloudinary

Option 3: Use Admin Dashboard
-----------------------------
1. Start the server
2. Go to Admin Dashboard > Settings
3. Upload images directly via the interface (coming in v1.1)

                           CLOUDINARY CONFIGURATION

✅ CLOUDINARY REQUIRED

Cloudinary is REQUIRED for image storage. Without it:
- Product images won't work
- Hero images won't work
- Logo won't display

To configure:
1. Sign up at https://cloudinary.com/
2. Get your Cloudinary URL from dashboard
3. Add to .env:
   CLOUDINARY_URL=cloudinary://api_key:api_secret@cloud_name
