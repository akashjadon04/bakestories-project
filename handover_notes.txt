                        IMAGE RIGHTS & LEGAL NOTICE
================================================================================

⚠️ IMPORTANT: You must confirm you have the right to use all images on this
website, including any images sourced from Instagram.

Instagram Images:
-----------------
If you are using images from your Instagram account (@thebakestories05),
you automatically have the right to use them on your website.

However, if you are using images from:
- Other Instagram accounts
- Third-party sources
- Stock photo websites

You MUST have explicit permission or a valid license to use those images.

Consent Checkbox:
-----------------
In the Admin Dashboard > Settings, there is a checkbox to confirm:
"I confirm I own the rights to all images used on this website"

Please check this box after reviewing all images.

                           CONFIGURATION CHECKLIST
================================================================================

Before going live, complete these steps:

1. IMAGES
   [ ] Upload your logo to Cloudinary or replace in code
   [ ] Upload product images to Cloudinary
   [ ] Upload hero carousel images (or connect Instagram)
   [ ] Confirm image rights checkbox in admin settings

2. CONTACT INFORMATION
   [ ] Update phone number in .env (BAKERY_PHONE)
   [ ] Update email in .env (BAKERY_EMAIL)
   [ ] Update business hours in footer
   [ ] Update address if applicable

3. PRODUCTS
   [ ] Review and update sample products
   [ ] Add your actual products with correct prices
   [ ] Set stock quantities
   [ ] Add product descriptions

4. PAYMENT & DELIVERY
   [ ] Confirm COD is your preferred payment method
   [ ] Set delivery charges (default: ₹50, free over ₹500)
   [ ] Define delivery areas
   [ ] Set delivery time slots

5. LEGAL
   [ ] Add privacy policy page
   [ ] Add terms of service page
   [ ] Add refund/cancellation policy
   [ ] Review and confirm image rights

6. TESTING
   [ ] Test complete order flow
   [ ] Test admin dashboard
   [ ] Test on mobile devices
   [ ] Test email/SMS notifications (if configured)
